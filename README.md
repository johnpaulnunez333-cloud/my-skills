# 🛠️ My Skills Showcase Web App

Isang interactive at modernong web-based dashboard na binuo upang ipakita ang aking kasalukuyang mga teknikal na kakayahan, mga pinag-aaralang teknolohiya, at direksyon bilang isang developer.

## 🚀 Live Demo
👉 https://johnpaulnunez333-cloud.github.io/my-skills/

## 💡 Paano Ko Ito Ginawa (Design & AI Collaboration)
Ang proyektong ito ay nagpapakita ng aking pagtutok sa frontend presentation at UI/UX design:
- Ginamit ko ang **Claude AI** bilang katuwang upang mag-validate ng maayos na layout at masigurong malinis ang visual hierarchy ng aking portfolio dashboard.
- Nakatutok ang proyektong ito sa aking pangunahing lakas sa **UI/UX design** at frontend components gamit ang **React (.jsx)** at **Vanilla JavaScript**.
- Manu-mano kong isinama ang aking core strengths habang idinodokumento ang aking kasalukuyang pag-aaral sa backend data management.

## ✨ Mga Tampok (Features)
- **Skill Roadmap Visualization:** Isang malinaw na pagpapakita ng aking mga alam na teknolohiya at mga kasalukuyang pinag-aaralan pa lang.
- **Frontend Focus:** Malakas na pagpapakita ng modern interface practices gamit ang React components (`react.jsx`).
- **Interactive UI:** Dynamic animations at layout adjustments na pinapatakbo ng `script.js` at custom `style.css`.

## 🧠 Kasalukuyang Core Skills
- **Napakalakas (Main Focus):** HTML5, CSS3, Vanilla JavaScript, React.js, at UI/UX Front-End Design.
- **Nagsisimula Pa Lang (Beginner / Learning):** SQL (Database Management) at Python Back-End Server Logics.

## 📁 Estruktura ng Repository
- `react.jsx` - Naglalaman ng modern component structures para sa maayos na pag-render ng mga skill cards.
- `script.js` - Nagpapatakbo ng mga native client-side interactivity at dynamic behaviors.
- `style.css` - Custom styling sheet para sa kulay, spacing, at pangkalahatang visual aesthetic ng website.
- `index.html` - Pangunahing entry point ng web app.
